Finalizado

O 80's Karaoke é um karaokê com estética anos 80 que roda no navegador. Atualmente, há 5 músicas disponíveis para teste. A página do 80's Karaoke se encontra aqui.

A exibição das letras na tela e o seu preenchimento sincronizado à música foi feito totalmente em JavaScript nativo e CSS. Para o playback de áudio, utilizei o howler.js.

O back-end responsável por disponibilizar as letras e os arquivos de áudio é uma API REST, desenvolvida em Java, utilizando o framework Jersey.

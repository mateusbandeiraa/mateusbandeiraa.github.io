QuestOne2018

Finalizado

O projeto de codinome JogoTrivia foi um trabalho para a faculdade desenvolvido em equipe, junto com Roberto Barbosa e Rafael Mota. O QuestOne é um jogo de perguntas e respostas multiplayer online. As perguntas aparecem em tempo real para todos os jogadores e os pontos são distribuídos de acordo com a velocidade que a resposta correta é selecionada.

A API do jogo foi desenvolvida em Java, e o cliente é uma página web que se comunica com o servidor via AJAX.
